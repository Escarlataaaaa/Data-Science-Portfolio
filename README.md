# Data Science Portfolio

It serves as a record of my technical work, problem-solving approach, and applied experience. Each project includes a **Jupyter notebook that walks through the code, analysis, and results**.



## Coursework

- **[Data 100 Projects](Data-100-projects/)**

## Research projects
